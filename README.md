Obiectiv: proiectarea unui lanț de procesare pentru semnale de mică amplitudine
Mod de implementare:
• Structurarea circuitului în patru etaje: amplificator de instrumentație, filtru activ trece‑jos, amplificator cu câștig
programabil (PGA) și redresor de precizie dublă alternanță.
• Implementare software: testarea integrității semnalului prin analize de tip Transient și AC Sweep în mediul
LTspice.
• Implementare hardware: realizarea circuitului pe breadboard și validarea performanțelor folosind instrumentele
de măsură virtuale din aplicația Scopy.
